Installation Instructions

 

Allison uses a standard WeiDU installer.  Once you've extracted the files into your main BG2 directory (the same

directory that contains dialog.tlk), run the setup-allison Executable which will be found in your BG2 folder. 

Simply answer the questions as they arise.

 

Note:  You must remove any previous version of Allison before installing V1.5.  Most importantly, delete setup-allison.tp2 from the BG2 directory before running the installer.

 

Mac OSX Specific Instructions:   If you have a previous version of Allison installed, it is necessary for you to uninstall it before installing the current version.  Please note that this means that you must remove any previous version of Allison before removing any files from the ZIP archive.

 

Once all files have been extracted to your root BG2 directory, double-click on setup-allison.command.  From here, follow the same instructions as the PC users.

 

 

 

Removing the Mod

 

Run the setup-allison executable.  You will be given the option to uninstall.  Note that this will not affect games already in progress.

 

 

Compatibility Concerns

 

This mod is compatible with all known WeiDU mods.  There should not be any issues if Allison is installed last.  I have tested it with, and verified compatibility with the following mods:

 

-Improved Asylum

-Kelsey

-Solaufein

-Oversight

-The Unusual Oddities Shop

-RPG Dungeon Item Pack

-Kitanya

-Flirt Packs

-Banter Packs

-Any portrait pack mod

-Unfinished Business

 

If you find a mod that does conflict with Allison, send an e-mail to:  rastor@rpgdungeon.net with "Allison" in the subject line and I'll either fix the problem or include a warning with the mod.
